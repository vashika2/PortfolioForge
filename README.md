# PortfolioForge - MERN Portfolio Builder

PortfolioForge is a full-stack portfolio builder that allows users to create, save, preview, share, and download professional portfolios.

## Features

- User registration and login
- JWT authentication
- Protected dashboard
- Profile picture upload
- Resume PDF upload
- Education section
- Experience section
- Skills badges
- Dynamic projects
- Project tech stack, GitHub link, and live demo link
- Save portfolio to MongoDB
- Public portfolio sharing URL
- Download portfolio as PDF
- Toast notifications
- Responsive UI using Tailwind CSS

## Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS
- Axios
- React Router
- React Hot Toast
- jsPDF
- html2canvas

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- bcryptjs
- JSON Web Token
